# EC_Automation
Guru99 E-commerce automation testing ptoject using selenium in java , maven , Testng and POM (project object model)
test cases covered:
1-Verify Create Account 
2-Verify Items can be sorted by "Name" 
3-Verify cost of product in list page and details page are equal 
4-Verify can't add more products in cart than the product available in store 
5-Verify compare between 2 products 
6-Verify share wishlist to other people using email 
7-Verify purchase product
8-Verify save previously placed orders as a pdf file
9-Verify change or reorder previously added products 
10-Verify Discount Coupon works correctly
